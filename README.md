# Employee Feedback System

Telegram-бот для сбора, хранения и модерации идей сотрудников компании.

Проект позволяет сотрудникам отправлять предложения по улучшению работы компании, отслеживать статус своих идей, а администраторам — просматривать и модерировать предложения через встроенную admin-панель.

---

# Функционал проекта

## Возможности пользователя

- Добавление новой идеи
- Подтверждение отправки идеи
- Просмотр списка своих идей
- Просмотр статуса конкретной идеи
- Получение уведомлений о модерации идеи

---

## Возможности администратора

- Просмотр всех предложенных идей
- Добавление новых администраторов
- Модерация идей
- Изменение статуса идеи
- Просмотр информации об авторах идей

---

# База данных

В проекте используется SQLite.

Создаются следующие таблицы:

## ideas

Таблица для хранения идей сотрудников.

### Поля:
- `id` — уникальный идентификатор идеи
- `text_idea` — текст идеи
- `chat_id` — Telegram ID пользователя
- `user_name` — username пользователя
- `created_at` — дата создания идеи
- `status` — статус идеи
- `moderator_comment` — комментарий администратора

---

## admin_user

Таблица для хранения администраторов.

### Поля:
- `id` — идентификатор администратора
- `user_name` — имя администратора
- `user_chat_id` — Telegram chat_id администратора
